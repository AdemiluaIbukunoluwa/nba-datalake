# NBA DATA-LAKE

## Overview
This system leverages Amazon S3 and AWS Glue to fetch data and prepare it for querying with Athena.

## Technologies used
- Programming Language: Python
- API: SportsData.io
- AWS Services:
    - Amazon S3: to store raw data fetched from the API
    - AWS Glue: Extracts and loads the data to prepare it for querying
    - Amazon Athena: To query data
